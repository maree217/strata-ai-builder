# Framework

**The "what" — a practical guide to implementing Strata in your organization.**

---

## Overview

The Strata Framework translates philosophy into action. It provides:

1. **Assessment** — Know where you are
2. **Layer Guides** — Know what to build
3. **Governance** — Know how to stay compliant

---

## Start Here: Assessment

Before implementing anything, understand your current state.

### Quick Assessment (60 seconds)

| Question | Yes | No |
|----------|-----|-----|
| Can you list all your AI initiatives right now? | ✅ | ❌ |
| Do you know the ROI of each? | ✅ | ❌ |
| Do you review performance monthly? | ✅ | ❌ |

**3 Yes:** You're ahead — focus on optimization
**1-2 Yes:** You have gaps — start with visibility
**0 Yes:** You're at risk — start with assessment

## Implementation Patterns

### Pattern A: Greenfield (New AI Capability)

```
You have: No AI infrastructure
You want: AI capability

Path:
1. Assessment (1 week)
2. Layer 1 build (4-8 weeks)
3. Layer 2 build (6-12 weeks)
4. Layer 3 build (4-8 weeks)
5. Governance throughout

Total: 4-7 months
```

### Pattern B: Brownfield (Existing AI Chaos)

```
You have: Scattered AI experiments
You want: Unified portfolio

Path:
1. Assessment + inventory (2 weeks)
2. Layer 1 retrofit (6-10 weeks)
3. Layer 2 standardization (8-12 weeks)
4. Layer 3 consolidation (4-8 weeks)
5. Governance remediation throughout

Total: 5-8 months
```

### Pattern C: Optimization (Mature AI)

```
You have: Working AI, want to improve
You want: Better ROI, lower risk

Path:
1. Assessment + gap analysis (1 week)
2. Layer 1 optimization (2-4 weeks)
3. Layer 2 enhancement (4-6 weeks)
4. Layer 3 refinement (2-4 weeks)
5. Governance hardening throughout

Total: 2-4 months
```

## Success Criteria

### Phase 1 Complete When:

- [ ] All AI costs are tracked and visible
- [ ] Security controls are implemented
- [ ] Audit trails are functioning
- [ ] Cost savings documented (target: 30-50%)

### Phase 2 Complete When:

- [ ] Data pipelines are automated
- [ ] Models are version-controlled
- [ ] Learning loops are operational
- [ ] ROI improvement documented (target: 2-4x)

### Phase 3 Complete When:

- [ ] User experiences deployed
- [ ] Adoption metrics tracked
- [ ] Feedback loops established
- [ ] Full ROI documented (target: 5-10x)

### Governance Complete When:

- [ ] All initiatives documented
- [ ] Risk assessments completed
- [ ] Review cadence established
- [ ] Compliance requirements met
